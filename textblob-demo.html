TextBlob Minimalist Demo

This demo showcases the core functionality of TextBlob, a Python library for processing textual data. Enter text in the box below to analyze it.

Sentiment Analysis

TextBlob can analyze the sentiment of text, returning polarity (negative to positive, -1.0 to 1.0) and subjectivity (objective to subjective, 0.0 to 1.0).

from textblob import TextBlob
blob = TextBlob("TextBlob is amazing!")
sentiment = blob.sentiment
print(f"Polarity: {sentiment.polarity}, Subjectivity: {sentiment.subjectivity}")

Parts of Speech Tagging

TextBlob can identify parts of speech in text (nouns, verbs, adjectives, etc.).

from textblob import TextBlob
blob = TextBlob("TextBlob is a simple library for processing text.")
pos_tags = blob.tags
print(pos_tags)

Translation

TextBlob can translate text between different languages (using Google Translate API).

from textblob import TextBlob
blob = TextBlob("Hello, how are you?")
translated = blob.translate(to='es')
print(f"Translated text: {translated}")

Server-Side Implementation

Here's the Python code you would need for a Flask server to implement this functionality:

from flask import Flask, request, jsonify from textblob import TextBlob import json app = Flask(__name__) @app.route('/analyze', methods=['POST']) def analyze(): data = request.get_json() text = data.get('text', '') if not text: return jsonify({'error': 'No text provided'}), 400 blob = TextBlob(text) # Sentiment analysis sentiment = blob.sentiment # Parts of speech pos_tags = [(word, pos) for word, pos in blob.tags] # Language detection try: language = blob.detect_language() except: language = "Unable to detect" # Spell check try: corrected = str(blob.correct()) except: corrected = "Spell check unavailable" # Translation (to Spanish as example) try: translated = str(blob.translate(to='es')) except: translated = "Translation unavailable" # Noun phrases noun_phrases = list(blob.noun_phrases) result = { 'sentiment': { 'polarity': sentiment.polarity, 'subjectivity': sentiment.subjectivity }, 'pos_tags': pos_tags, 'language': language, 'corrected': corrected, 'translated': translated, 'noun_phrases': noun_phrases } return jsonify(result) if __name__ == '__main__': app.run(debug=True)
